Hijri Date

Overview

This add-on enhances NVDA's date reporting by adding the Hijri (Islamic) date alongside the Gregorian date.

Usage

  • NVDA+F12 (press once): Reports the current time.
  • NVDA+F12 (press twice): Reports both the Gregorian and Hijri dates.

Settings

You can configure the add-on from NVDA Preferences > Settings > Hijri Date:

  • Date announcement priority: Choose whether the Hijri date or Gregorian date is announced first.
  • Check for updates: Check for new versions of the add-on from GitHub.

Updates

The add-on automatically checks for updates silently in the background. You can also check manually from the settings panel. When an update is available, you will see the changelog and can download and install the update with a single click.

Date Format

The Gregorian date is displayed using your Windows system locale format. The Hijri date is displayed as: day month, year AH (e.g. 11 Ramadan, 1448 AH).

Hijri Calendar

The add-on uses the official Umm al-Qura calendar data for accurate Hijri date conversion, with a fallback algorithm for dates outside the supported range.

Compatibility

This add-on requires NVDA 2023.1 or later.
